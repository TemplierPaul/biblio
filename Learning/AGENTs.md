# Learning Directory: Agent Navigation Guide

This directory contains resources and notes on various disciplines related to optimization, game theory, and machine learning.

## Subdirectories

- **[Evolutionary Optimization](file:///Users/ptemplie/Documents/research_workspace/Learning/Evolutionary%20Optimization)**: Focused on Quality-Diversity (QD), open-endedness, and evolutionary algorithms.
- **[Game Theory](file:///Users/ptemplie/Documents/research_workspace/Learning/Game%20Theory)**: Fundamentals of game theory, equilibrium concepts, and multi-agent systems.
- **[JAX Tools](file:///Users/ptemplie/Documents/research_workspace/Learning/JAX%20Tools)**: Performance-oriented machine learning with JAX, JIT practices, and ecosystem details.
- **[Machine Learning](file:///Users/ptemplie/Documents/research_workspace/Learning/Machine%20Learning)**: Core ML concepts including Transformers, LLMs, GNNs, and optimization losses.
- **[Reinforcement Learning](file:///Users/ptemplie/Documents/research_workspace/Learning/Reinforcement%20Learning)**: RL methods for PSRO, GRPO, continual RL, and world models.
